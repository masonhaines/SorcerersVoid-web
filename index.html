Sorcererâ€™s Void is a challenging action RPG in the Souls-like genre, developed in Unreal Engine 5. The game blends familiar mechanics with new elements to create a fresh and engaging experience.

Set in a medieval world, players navigate shifting landscapes, floating terrains, and mysterious portals, encountering formidable enemies with distinct attack patterns. Combat emphasizes fluid mechanics, including combos, dodging, parrying, and cinematic executions, requiring a strategic approach. Enemies utilize AI-driven behavior trees, creating dynamic and responsive encounters.

Exploration plays a key role, with hidden items and an inventory system aiding progression. A gem-based progression system enhances stats such as strength, defense, health, and stamina.

Key Features:

  • Player Combat System - Multiple weapons, blocking, sprinting, and dodging.
  • Enemy AI System - Behavior trees drive dynamic enemy actions.
  • Character Progression - Upgrade stats through experience and special gems.
  • Inventory & Loot System - Manage weapons, armor, and consumables.
  • Tutorial System - Introduces core combat and movement mechanics.

Built using C++ and Unreal Engine Blueprints, the game integrates AIModule, NavigationSystem, AnimGraphRuntime, and EnhancedInput. Upon completion, it will be packaged for distribution on GitHub.
